# ds-salaries-timeseries
# Прогнозирование зарплат в Data Science

Этот проект представляет собой комплексный анализ и прогнозирование зарплат специалистов в области Data Science на основе датасета с Kaggle.

**Ключевая особенность:** В отличие от большинства решений, здесь используется **честная временная валидация (TimeSeriesSplit)**, исключающая утечку данных из будущего в прошлое. Модели обучаются на данных 2020-2021 годов и тестируются на данных 2022 года.

## 📊 Используемые технологии
- Python (pandas, numpy, matplotlib, seaborn)
- Scikit-learn (Pipeline, ColumnTransformer, различные модели и метрики)
- Category encoders (OneHotEncoder, TargetEncoder)

## 📈 Основные этапы работы
1.  **Разведочный анализ:** Изучение распределений и влияния категориальных признаков.
2.  **Предобработка:** Разделение данных по годам, обработка пропусков (их нет), кодирование признаков с учётом их природы (OneHot, Target, Ordinal).
3.  **Валидация:** Применение `TimeSeriesSplit` для кросс-валидации.
4.  **Моделирование:** Сравнение Lasso-регрессии и стекинг-модели (RandomForest + Lasso с мета-оценщиком Ridge).
5.  **Анализ важности признаков:** Интерпретация результатов.

## ✅ Результаты
Итоговая стекинг-модель показала следующие результаты на тестовой выборке (2022 год):
- **R² ≈ 0.5**
- **RMSE ≈ 270k USD**
- **MAE ≈ 52k USD**

**Что это значит?** Модель не идеальна (что нормально для зарплат), но она **честно обучена** и дает адекватные прогнозы (исходный датасет имеет дисбаланс в тренировочной и тестовой выборке), не переобучаясь. Ее можно использовать для анализа тенденций и базового прогнозирования.

## 📁 Данные
Исходный датасет: [Data Science Job Salaries](https://www.kaggle.com/datasets/ruchi798/data-science-job-salaries) на Kaggle.
